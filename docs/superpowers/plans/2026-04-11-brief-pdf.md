# Brief PDF — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ajouter deux boutons de téléchargement PDF dans l'onglet Brief du CRM — un formulaire vierge à envoyer au client et un récapitulatif des réponses reçues.

**Architecture:** Génération 100% client-side via `@react-pdf/renderer`. Un composant `BriefPDFDocument` à deux modes (`blank` / `filled`) wrappé par `PDFDownloadLink` dans `ProjectBrief.tsx`. Le hook `useBriefV2` est étendu pour retourner le nom du projet (déjà fetchée depuis `projects_v2`).

**Tech Stack:** `@react-pdf/renderer` v4, React 18, TypeScript 5, Vite 5.
